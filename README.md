# Air Flight Booking System ✈️

This is a C++ console-based project.

## Features:
- Add flight booking
- View bookings
- Update booking
- Delete booking (CRUD operations)
- Seat and food selection system

## Language Used:
C++

## Purpose:
This project is made for learning basic programming concepts like structures, arrays, loops, and functions.
